ExactCN: Predicting Exact Copy Numbers on Whole Exome Sequencing Data

Erfan FarhangKia, Ahmet Arda Ceylan, Mert Gencturk, Mehmet Alper Yilmaz, Furkan Karademir, A. Ercument Cicek

ExactCN is a deep learning method for predicting exact integer copy numbers at the exon level from whole-exome sequencing (WES) data using read-depth signals. The model processes base-level WES read-depth signals through convolutional layers and Performer encoder blocks to regress predicted copy numbers. Trained on 530 WES samples from the 1000 Genomes Project, ExactCN achieves state-of-the-art performance with 32% MAE reduction, improving macro-averaged MAE from 0.91 to 0.62 and RMSE from 1.31 to 0.78.

Bridging Local and Federated Data Normalization in Federated Learning: A Privacy-Preserving Approach

Mert Gencturk, et al.

This paper addresses data normalization challenges in federated learning (FL), where data remains distributed across multiple parties. We introduce "federated normalization," which enables collaborative exchange of normalization parameters to achieve performance comparable to pooled normalization while preserving data locality. Key contributions include systematic evaluation of federated and local normalization techniques in heterogeneous FL scenarios, a novel encryption method for calculating k-th ranked elements in federated settings, and privacy-preserving implementations using multiparty fully homomorphic encryption (MHE).
