Developed and implemented a comprehensive Internship Management System for Bilkent University's Computer Science Department, currently serving over 500 users since September 2023.

**Technologies Used:**
- Backend: Java, Spring Framework, MongoDB, Redis
- Frontend: React
- Infrastructure: AWS (EC2, S3, ElastiCache)

**Key Contributions:**
- Led requirement analysis and authored comprehensive technical reports with UML diagrams
- Collaborated with peers on system architecture and implementation
- Managed deployment using AWS services for scalability and performance
- Oversaw a new development team, conducting code reviews and providing technical guidance

**Impact:**
- Successfully replaced manual internship tracking processes
- Streamlined communication between students, faculty, and administration
- Improved efficiency in internship approval and documentation workflows
- Recognized with the Technical Service Award from Bilkent University in 2024

This project demonstrated my ability to lead full-stack development projects from requirements gathering through deployment and maintenance, while mentoring other developers and ensuring code quality through rigorous review processes.
